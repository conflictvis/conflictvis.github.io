API: allows a user's systems (database, scripts) to interface directly with the visualization/webpage's system in order to extract and process that data.
updated: the visualization is actively maintained; thus, the data and visualization is updated every year, month, week, etc.
save image: the full visualization image can be directly downloaded to a user's computer
interactive: allows user interaction beyond hovering and abstraction, such as filtering or reconfiguration.
data available: the data source used by the visualization can be directly downloaded from the webpage via a download button, link to data source, or API.
